Class: LegendItem

LegendItem

A legendItem is used to add text to a map legend. Individual Features and ArcPlots can be linked to a legendItem, so that the feature or arcPlot color will use the swatchColor of legendItem.

new LegendItem(legend, data, meta)

LegendItem.js, line 11

A legendItem is used to add text to a map legend. Individual Features and ArcPlots can be linked to a legendItem, so that the feature or arcPlot color will use the swatchColor of legendItem.

Name Type Description
legend Legend

The parent Legend for the LegendItem.
data Object

Data used to create the legendItem:

Option Default Description
text '' Text to display
drawSwatch false Should a swatch be drawn beside the text
font Legend font A string describing the font. See Font for details.
fontColor Legend fontColor A string describing the color. See Color for details.
textAlignment Legend textAlignment left, center, or right
swatchColor 'black' A string describing the color. See Color for details.
swatchOpacity 1 A value between 0 and 1.
meta Object optional

User-defined key:value pairs to add to the legendItem.
