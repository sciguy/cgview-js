# Plasmid
The map above can be moved around by dragging and zoomed by using your touchpad/mouse scroll. The legend colors can be changed by clicking on the swatches. The viewer object for this map can be accessed in your browser JavaScript console as cgv. Note, that the controls below the map are custom controls created in the tutorial Map Controls.
